
## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## ENV Variables

|Var name     | Default value | Description           |
|-------------|---------------|-----------------------|
| PG_HOST     | localhost     | PostgresSQL host      |
| PG_PORT     | 5432          | PostgresSQL port      |
| PG_USER     | nest_learning | PostgresSQL username  | 
| PG_PASSWORD | qwerty        | PostgresSQL password  |
| PG_DATABASE | gis_db        | Database name         |
| PORT        | 3000          |  app port             |
|MODE         | PRODUCTION    | App Mode (not using for now) |



## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
